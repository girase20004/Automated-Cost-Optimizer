# Automated-Cost-Optimizer
Automated AWS cost optimization project using Lambda and CloudWatch to automatically stop unused EC2 instances and reduce cloud costs.


## 🎯 Purpose
Automatically reduce AWS costs by stopping unused EC2 instances.

## 🧰 AWS Services Used
- AWS Lambda
- Amazon EC2
- Amazon CloudWatch
- AWS IAM

## 📌 Project Overview
This project automates AWS cost optimization using Lambda and CloudWatch. 
Unused EC2 instances are automatically stopped based on schedules or monitoring rules.

## 🚀 Features
- Automated cost saving
- Serverless automation
- Scheduled monitoring
- Reduced manual effort
